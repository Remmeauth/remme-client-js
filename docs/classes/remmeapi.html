Class RemmeApi

Main class that send requests to our REMME protocol; Check JSON-RPC API specification: https://remmeio.atlassian.net/wiki/spaces/WikiREMME/pages/292814862/RPC+API+specification.

example
import { RemmeApi, RemmeMethods } from "remme-api";

const remmeApi = new RemmeApi({
     nodeAddress: "localhost",
     nodePort: 8080,
     sslMode: false,
});

const response = await remmeApi.sendRequest<object>(RemmeMethods.fetchBlocks);
console.log(response);
